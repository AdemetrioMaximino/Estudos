  • Para criar um código inicial é simples. Escreve "!" e aperta ENTER.
  • A maioria das tags pode ser escritas sem "<>", e após apertar ENTER, são colocadas automáticamente, e se precisar fechar a tag, também é fechado automáticamente.
  • Pra facilitar em colocar tag em um texto já escrito, só precica selecionar, apertar CTRL + ALT + P, e selecionar a opcão Emmet: Warp with Abbreviation (Envolver com Abreviação), após isso, só colocar a tag.
  • Lembrar sempre de manter a opção Word Wrap ativa, para quebrar linha automáticamente
